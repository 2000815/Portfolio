SKILL

HTML5/CSS3

HTMLとCSSという言語でほとんどのwebページの見た目が作られています。HTMLは、文書を構造化するための言語で マークアップ言語といいます。 CSSは、作成されたHTMLの見た目や装飾を整えるもので スタイルシート言語ともいいます。

Python

今最も注目を集めているプログラミング言語です。その特長として、人工知能（機械学習や深層学習）などの最先端分野の開発にも使われている一方、文法が平易なため、 大規模なWebアプリケーション開発が短期間で作ることが可能です。

JavaScript

JavaScriptとは、動的なWebページを作成する事のできるプログラミング言語です。JavaScriptを使うと、ユーザーのアクションに応じたコンテンツの表示の他、ブラウザー上で表示される地図やグラフィックアニメーションなども表示する事ができます。

Django

Djangoは、安全でメンテナンス可能なWebサイトの迅速な開発を可能にする、高度なPython Webフレームワークです。 経験豊富な開発者によって開発されたDjangoは、Web開発の多くの面倒事を引き受けてくれます。

Bootstrap

BootstrapはCSSのフレームワークです。このフレームワークにはよく使われるスタイルがあらかじめ定義してあるので、ルールに沿って利用するだけで整ったデザインのページを作成できます。

Responsive
